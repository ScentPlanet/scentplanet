# Contributing to ScentPlanet

Thank you for your interest in contributing to ScentPlanet! While the core application is proprietary, there are several ways you can help improve the project.

## Ways to Contribute

### 1. Game Profiles (Recommended)

The best way to contribute is by adding support for new games! Our game profile library is open source and welcomes community contributions.

**[ScentPlanet/game-profiles](https://github.com/ScentPlanet/game-profiles)**

Each game profile defines:
- Window detection patterns
- Environment types and scent mappings
- Custom intensity settings

See the [game-profiles README](https://github.com/ScentPlanet/game-profiles#readme) for detailed instructions.

### 2. Bug Reports

Found a bug? Please report it!

1. Check [existing issues](https://github.com/ScentPlanet/scentplanet/issues) to avoid duplicates
2. Use our [bug report template](https://github.com/ScentPlanet/scentplanet/issues/new?template=bug_report.yml)
3. Include as much detail as possible

### 3. Feature Suggestions

Have an idea to improve ScentPlanet?

1. Check [existing feature requests](https://github.com/ScentPlanet/scentplanet/issues?q=is%3Aissue+label%3Aenhancement)
2. Use our [feature request template](https://github.com/ScentPlanet/scentplanet/issues/new?template=feature_request.yml)
3. Describe the use case and proposed solution

### 4. Translations

Help us reach more users by improving translations:

- English (en)
- Simplified Chinese (zh-CN)
- Traditional Chinese (zh-TW)

Contact us on [Discord](https://discord.gg/scentplanet) if you'd like to help with translations.

### 5. Documentation

Help improve our documentation:

- Fix typos or unclear explanations
- Add examples or tutorials
- Translate documentation

Submit changes as a Pull Request to this repository.

## Code of Conduct

Please note that this project is released with a [Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.

## Questions?

Join our [Discord server](https://discord.gg/scentplanet) for help and discussion!
